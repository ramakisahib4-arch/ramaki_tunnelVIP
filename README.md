# P2P Tunnel - برنامه بومی انتقال اینترنت

یک برنامه اندرویدی بومی خالص که اینترنت را مستقیماً بین دو دستگاه منتقل می‌کند.

## ویژگی‌ها
- ✅ بومی خالص (بدون WebView، بدون HTML)
- ✅ اتصال Socket مستقیم P2P
- ✅ هر لینک منحصربه‌فرد و غیرقابل تکرار
- ✅ مصرف اینترنت از Provider (رفیق)

## نحوه استفاده
### Provider (دهنده):
1. دکمه سبز "ساخت لینک و شروع" را بزنید
2. لینک ساخته شده را برای رفیق بفرستید
3. منتظر اتصال بمانید

### Consumer (گیرنده):
1. لینک رفیق را در کادر بگذارید
2. دکمه آبی "اتصال به اینترنت رفیق" را بزنید
3. اینترنت از رفیق مصرف می‌شود
